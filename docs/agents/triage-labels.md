# Triage Labels

The skills speak in terms of five canonical triage roles. This file maps those roles to the Taskline task labels used in this repo.

| Label in mattpocock/skills | Taskline label | Meaning                                      |
| -------------------------- | -------------- | -------------------------------------------- |
| `needs-triage`             | `needs-triage` | Maintainer needs to evaluate this task       |
| `needs-info`               | `needs-info`   | Waiting on reporter for more information     |
| `ready-for-agent`          | `ready-for-agent` | Fully specified, ready for an AFK agent   |
| `ready-for-human`          | `ready-for-human` | Requires human implementation or decision |
| `wontfix`                  | `wontfix`      | Will not be actioned                         |

When a skill mentions a role, use the corresponding Taskline label. Preserve unrelated labels when changing triage labels.
